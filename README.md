# 🎂 Rainbow Birthday Card Generator

A fun and colorful web application that generates personalized birthday cards with hilarious birthday messages based on a person's **name**, **age**, and **hobby**.

## ✨ Features

* 🌈 Animated rainbow background
* 🎉 Festive birthday-themed design
* 🎂 Personalized birthday cards
* 😂 Random funny birthday messages
* 📝 User inputs name, age, and hobby
* 📚 Card history is preserved (new cards stack on top)
* 📱 Fully responsive design
* ⚡ Built with pure HTML, CSS, and JavaScript
* 🚫 No external libraries required

## 🚀 How It Works

1. Enter the person's **Name**
2. Enter their **Age**
3. Enter their **Hobby**
4. Click **Generate Birthday Card**
5. Receive a personalized and funny birthday message inside a beautifully designed birthday card

Each generated card remains visible, creating a collection of birthday wishes.
